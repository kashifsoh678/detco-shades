# Detco Shades Systems

Professional website for Detco, a leading sun shades and tensile structures company in Saudi Arabia.
Built with Next.js 16, Tailwind CSS, and TypeScript.

## Features

- **Responsive Design**: Mobile-friendly layout using Tailwind CSS.
- **Product Catalog**: 15+ product categories with dynamic detail pages.
- **Project Gallery**: Showcase of completed engineering projects.
- **Client List**: Grid of major partners and clients.
- **Contact Form**: Functional layout for inquiries with Google Maps integration.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
